# Licence Management Scripts

PowerShell scripts for assigning, removing and auditing Microsoft 365 licences for users and groups.

- `Get-M365LicencesReport.ps1`  
  Generates a full Microsoft 365 licensing report including total purchased licences,
  consumed licences, unassigned licences and per-user licence assignments. Useful for
  audits, cost optimisation and identity lifecycle governance.
